<h1 align="center">Dar AL-NAWATI - frontend</h1>

### Installation

```bash
$ npm install

# Development
$ npm run dev

# Production
$ npm run build
$ npm preview
```

#### [!CAUTION] You need to add the necessary environment variables

```env
VITE_API_BASE_URL="https://your_backend.uri"
```
